## Tech stack

### Front-end

Framework: NextJS

Language: Typescript

Design: shadcn

Styling: TailwindCSS

Code quality: ESLint, Prettier

Locale: next-intl

### Back-end

Framework: Spring Boot

Language: Java

Dependencies:

- Security
- Validation
- Flyway
- JPA data

### Database (containerized)

RDBMS: PostgreSQL

GUI: pgAdmin 4

> NOTE: To run these locally: `docker compose up -d`
> Check out the `compose.yml` file for credentials
